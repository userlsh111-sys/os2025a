1. 커밋 제거: git reset hard head~1(최근 1개)
2. --amend -m (새 커밋명): 커밋 이름 바꾸기
3. 로컬과 원격에서 내용을 다르게 수정했을 때(git)
-push: error남.
-pull: 병합으로, 그 내용이 든 파일이 CONFLICT됐음을 알려줌.
-cat: pull하고 입력하면 그 내용을 알려줌.
-그 내용을 수정하고, 커밋하면 결과적으로 같은 커밋으로 끝남.
4. git fetch: 원격의 커밋들을 로컬로 가져옴.
-pull을 누르면 원격에서 변경된 내용도 로컬에 반영됨.
--커밋이 원격으로 돌아감.
5. 깃(vi)으로 c 만들기: i(==insert== 떠야 입력 가능함.), :wq
6. git push -f origin (branch명): 원격을 로컬에 있는 걸로 완전히 대체해 버림.
